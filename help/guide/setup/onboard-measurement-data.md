# 添加和管理测量数据

{{limited-availability-release-note}}

本文档概述了将促销活动测量数据添加到Adobe Real-Time CDP Collaboration的步骤。 发布者可与Adobe团队合作，上传促销活动测量数据。 在上传和处理该数据后，发布商和广告商将能够查看大量的[促销活动测量报告](/help/guide/collaborate/measure.md)。
